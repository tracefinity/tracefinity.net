Tracefinity

Photograph your tools on paper, trace their outlines with AI, and generate 3D-printable Gridfinity bins to organise them.

Source Docker Gridfinity

How it works

1

Photograph

Place tools on A4 or Letter paper and photograph from above.

2

Corners

Adjust the paper corners so the app knows the real-world scale.

3

Trace

AI traces tool outlines into precise silhouettes via Gemini.

4

Generate

Arrange tools in a Gridfinity bin and export an STL for printing.

Quick start

docker run -p 3000:3000 \
  -v ./data:/app/storage \
  -e GOOGLE_API_KEY=your-key \
  ghcr.io/tracefinity/tracefinity

Details

  • Single Docker container: Next.js frontend + FastAPI backend
  • AI tracing via Google Gemini (gemini-3-pro-image-preview)
  • Manual mask upload as an alternative to API tracing
  • STL generation with build123d and the Gridfinity library
  • Tool library and bin designer with drag-and-drop layout
  • Automatic bin splitting for large grids that exceed bed size
